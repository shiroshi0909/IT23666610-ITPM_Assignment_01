Option 1: Transliteration Accuracy Testing (for students familiar with Sinhala) 
This assessment should be attempted individually. The assessment is worth 18% of the overall marks for the unit. This assignment is designed to evaluate your ability to test a live 
application, design automation strategies, and analyze its weaknesses in a structured manner. 
 
➢ Objective: 
• The main objective of this assignment is to assess how accurately the application available at the following link converts chat-style Singlish input into Sinhala output: 
https://www.pixelssuite.com/chat-translator 
• Accordingly, you are required to evaluate the correctness of the Chat Sinhala transliteration function by testing it with words, phrases, and sentence structures commonly 
used in informal Sinhala typing. Note that evaluating the Standard Sinhala transliteration function, backend APIs, and conducting performance, scalability, or security 
testing are outside the scope of this assignment.  
 
➢ Tasks: 
• Identify fifty test cases in which the system fails to correctly convert chat-style Singlish into Sinhala. These fifty scenarios must include at least two test cases for each of the 
twenty-four Singlish input types specified in Appendix 1. The remaining two test cases may represent any of the Singlish input types. Your test cases should not include 
any of the examples provided in Appendix 1 or Appendix 2. 
• Automate all identified scenarios using Playwright and record the execution results in the Excel file titled “Assignment 1 - Test cases.” Refer to Appendix 2 for guidance on 
how to complete the Excel file. The steps for automating the identified scenarios using Playwright are provided in the “Automation Steps - Option 1” document. In addition, 
the “test_automation” folder contains the files required to run Playwright and generate the results.
